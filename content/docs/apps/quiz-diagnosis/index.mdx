3つの質問に答えると、結果が変わる診断アプリを作ります。

このページでは、ただコードを写すのではなく、次の流れで「診断の仕組み」を作ります。

1. 質問と選択肢を設計する
2. `select.value` で選ばれた答えを読む
3. 答えに応じて点数を増やす
4. 点数によって結果を分ける
5. テーマを変えて自分の診断にする

---

```js
const q1 = document.querySelector('#q1');
const q2 = document.querySelector('#q2');
const q3 = document.querySelector('#q3');

const button = document.querySelector('#diagnoseButton');
const resultBox = document.querySelector('#resultBox');
const resultTitle = document.querySelector('#resultTitle');
const resultComment = document.querySelector('#resultComment');

button.addEventListener('click', () => {
    let score = 0;

    if (q1.value === 'active') {
        score++;
    }

    if (q2.value === 'active') {
        score++;
    }

    if (q3.value === 'active') {
        score++;
    }

    resultBox.className = 'result-box';

    if (score <= 1) {
        resultTitle.textContent = 'のんびりタイプ';
        resultComment.textContent = '家でゆっくり過ごすと元気になれそうです。';
        resultBox.classList.add('relax');
    } else if (score === 2) {
        resultTitle.textContent = 'バランスタイプ';
        resultComment.textContent = '休む時間と遊ぶ時間のバランスを取るのが得意です。';
        resultBox.classList.add('balance');
    } else {
        resultTitle.textContent = 'アクティブタイプ';
        resultComment.textContent = '外に出て動くことでリフレッシュできそうです。';
        resultBox.classList.add('active');
    }
});
```

<Section title="診断のルールを先に設計する" goal="HTMLを書く前に、どの答えをどう判定するか決めます。">

診断アプリでは、見た目より先に「判定ルール」を決めます。

今回は、答えを次の2種類に分けます。

- ゆっくり過ごす答え
- 活動的に過ごす答え

JavaScriptで判定しやすいように、それぞれの答えに短い名前を付けます。

| 答えの種類 | JavaScriptで使う短い名前 |
| --- | --- |
| ゆっくり過ごす答え | `relax` |
| 活動的に過ごす答え | `active` |

今回の判定ルールは、活動的な答えを選んだ数で結果を分ける形にします。

| 活動的な答えの数 | 結果 |
| --- | --- |
| 0〜1個 | のんびりタイプ |
| 2個 | バランスタイプ |
| 3個 | アクティブタイプ |

<Section title="HTMLで質問フォームを作る" goal="質問、選択肢、結果表示エリアを用意します。">

3つの質問は、同じ形で作ります。

質問の選択肢は、`<select>` と `<option>` で作ります。  
`<select>` は、複数の選択肢を1つの選択ボックスにまとめるフォーム部品です。  
`<option>` は、`<select>` の中にある1つの選択肢です。

各 `<option>` の `value` 属性には、さきほど決めた短い名前を書きます。  
たとえば、活動的な答えには `value="active"` を付けます。

<Action title="HTMLを書く">

`<body>` の中に次のHTMLを書きます。

```html
<main class="app">
    <h1>休日タイプ診断</h1>
    <p>質問に答えて、あなたの休日タイプを診断します。</p>

    <section class="question">
        <label for="q1">Q1. 休日の朝は？</label>
        <select id="q1">
            <option value="relax">ゆっくり寝たい</option>
            <option value="active">早起きして出かけたい</option>
        </select>
    </section>

    <section class="question">
        <label for="q2">Q2. 好きな過ごし方は？</label>
        <select id="q2">
            <option value="relax">家でのんびり</option>
            <option value="active">外で活動する</option>
        </select>
    </section>

    <section class="question">
        <label for="q3">Q3. 友達に誘われたら？</label>
        <select id="q3">
            <option value="relax">予定が合えば行く</option>
            <option value="active">すぐ行きたい</option>
        </select>
    </section>

    <button id="diagnoseButton">診断する</button>

    <section id="resultBox" class="result-box">
        <h2 id="resultTitle">診断結果</h2>
        <p id="resultComment">ボタンを押すと結果が表示されます。</p>
    </section>
</main>
```

</Action>

<Concept title="value 属性の役割">

画面に表示される文字と、JavaScriptで使う短い名前は別にできます。

```html
<option value="active">早起きして出かけたい</option>
```

この場合、画面には「早起きして出かけたい」と表示されます。  
JavaScriptで使う短い名前は `active` です。

</Concept>

<Action title="CSSとJavaScriptを読み込む">

`index.html` の `<head>` の中に、CSSとJavaScriptの読み込みを書きます。`script` に `defer` を付けると、HTMLを読み込み終えてからJavaScriptが実行されます。

```html
<head>
    <link rel="stylesheet" href="style.css">
    <script src="script.js" defer></script>
</head>
```

`style.css` と `script.js` は、このあとのステップで作ります。

</Action>

<Section title="選ばれた答えを読み取る" goal="選ばれている選択肢の短い名前を取得します。">

HTMLで作った部品をJavaScriptから操作するには、まず要素を取得します。

<Action title="要素を取得する">

`script.js` に次のコードを書きます。

```js
const q1 = document.querySelector('#q1');
const q2 = document.querySelector('#q2');
const q3 = document.querySelector('#q3');

const button = document.querySelector('#diagnoseButton');
const resultBox = document.querySelector('#resultBox');
const resultTitle = document.querySelector('#resultTitle');
const resultComment = document.querySelector('#resultComment');
```

</Action>

<Concept title="選ばれている値を読む">

`document.querySelector('#q1')` は、`id="q1"` が付いた `<select>` 要素を取り出します。  
その取り出した要素を、`q1` という変数に入れています。

`q1.value` は、いま選ばれている `<option>` の `value` 属性を読み取ります。  
取り出される値は `"relax"` や `"active"` のような文字列で、画面に表示されている文字ではありません。

たとえば、「早起きして出かけたい」が選ばれているとき、`q1.value` は `"active"` を返します。

</Concept>

<Action title="ボタンを押したときに値を確認する">

続けて、次のコードを書きます。

```js
button.addEventListener('click', () => {
    console.log(q1.value);
    console.log(q2.value);
    console.log(q3.value);
});
```

</Action>

<Verify title="値を確認する">

質問の答えを変えてから「診断する」ボタンを押し、コンソールに `relax` または `active` が表示されれば成功です。

</Verify>

<Recovery title="undefined やエラーが出るとき">

HTML側の `id` と、JavaScriptの `document.querySelector(...)` の名前が一致しているか確認します。  
特に `#diagnoseButton` の大文字小文字に注意してください。

</Recovery>

</Section>

<Section title="点数を計算する" goal="活動的な答えを選んだ数だけ点数を増やします。">

診断結果を決めるために、`score` という変数を使います。  
`score` は、選んだ答えをもとにした点数です。  
最初は0にして、`active` が選ばれていたら1ずつ増やします。

<Action title="score を計算する処理を書く">

クリックイベントの中を次のように書き換えます。

`q1.value` は文字列なので、`'active'` と比べます。  
`'active'` はクォートで囲んだ文字列リテラルで、変数名ではありません。

```js
button.addEventListener('click', () => {
    let score = 0;

    if (q1.value === 'active') {
        score++;
    }

    if (q2.value === 'active') {
        score++;
    }

    if (q3.value === 'active') {
        score++;
    }

    console.log(score);
});
```

</Action>

<Section title="点数で結果を切り替える" goal="score の値に応じて、表示する結果を変えます。">

`score` が計算できたら、次は結果表示です。  
文字だけでなく、結果ボックスの色も一緒に変えます。

<Concept title="結果の色を毎回付け直す">

結果を切り替えるたびに、前回の色クラスを消す必要があります。  
そのため、まず `className` で基本クラスだけに戻します。

```js
resultBox.className = 'result-box';
```

その後、結果に合うクラスを追加します。

```js
resultBox.classList.add('active');
```

</Concept>

<Action title="判定処理を追加する">

クリックイベントの中の `console.log(score);` を削除し、次の判定処理を追加します。

```js
    resultBox.className = 'result-box';

    if (score <= 1) {
        resultTitle.textContent = 'のんびりタイプ';
        resultComment.textContent = '家でゆっくり過ごすと元気になれそうです。';
        resultBox.classList.add('relax');
    } else if (score === 2) {
        resultTitle.textContent = 'バランスタイプ';
        resultComment.textContent = '休む時間と遊ぶ時間のバランスを取るのが得意です。';
        resultBox.classList.add('balance');
    } else {
        resultTitle.textContent = 'アクティブタイプ';
        resultComment.textContent = '外に出て動くことでリフレッシュできそうです。';
        resultBox.classList.add('active');
    }
```

</Action>

### 演習5

<Exercise>

診断アプリで起きやすいミスを見つけます。次のコードには、よくあるミスが4つ入っています。

```js
const q1 = document.querySelector('#q1');
const q2 = document.querySelector('#q2');
const q3 = document.querySelector('#q3');
const button = document.querySelector('#diagnoseButton');
const resultBox = document.querySelector('#resultBox');
const resultTitle = document.querySelector('#resultTitle');

button.addEventListener('click', () => {
    let score = 0;

    if (q1.value = 'active') {
        score++;
    }

    if (q2.value === active) {
        score++;
    }

    if (q3.value === 'active') {
        score++;
    }

    resultBox.classList.add('result-box');

    if (score = 2) {
        resultTitle.textContent = 'バランスタイプ';
    }
});
```

<Solution>

1. `q1.value = 'active'` は代入なので、`q1.value === 'active'` に直す。
2. `active` がクォートで囲まれていないので、`'active'` に直す。
3. `resultBox.classList.add('result-box')` では前回の色クラスが消えないので、`resultBox.className = 'result-box';` にする。
4. `if (score = 2)` も代入なので、`if (score === 2)` に直す。

</Solution>

</Exercise>

JavaScriptでは、q1、q2、q3 のすべてで同じように点数を足します。

```js
let score = 0;

if (q1.value === 'high') {

if (q2.value === 'high') {
```

```js
if (score >= 5) {
```
