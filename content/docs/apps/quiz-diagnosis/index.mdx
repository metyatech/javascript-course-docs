3つの質問に答えると「のんびりタイプ」「バランスタイプ」「アクティブタイプ」のどれかに診断されるアプリを作ります。`select` の値を読み取り、`if`/`else if` で結果とクラスを切り替える流れを確認します。

<Section title="完成形を確認する" goal="完成したアプリの動作と、登場する新しい記法を先に確認します。">

完成すると、3つの質問に答えてボタンを押すと、結果タイトルとコメントが表示され、タイプごとに結果ボックスの色が変わります。

| 操作 | 画面の変化 |
| --- | --- |
| 各 `<select>` で答えを選ぶ | 選んだ値が保持される |
| 「診断する」ボタンを押す | 結果タイトル／コメントが表示され、タイプごとに背景色が変わる |

今回新しく使う記法は次のとおりです。

| 記法 | 役割 |
| --- | --- |
| `select` | ドロップダウンで答えを選ばせるHTML要素 |
| `select.value` | 選ばれている選択肢の値を取り出す |
| `if` / `else if` / `else` | 条件によって処理を分ける |
| `classList.add(...)` | 要素にクラスを追加して見た目を切り替える |

```js
const q1 = document.querySelector("#q1");
const q2 = document.querySelector("#q2");
const q3 = document.querySelector("#q3");

const button = document.querySelector("#diagnoseButton");
const resultBox = document.querySelector("#resultBox");
const resultTitle = document.querySelector("#resultTitle");
const resultComment = document.querySelector("#resultComment");

button.addEventListener("click", function () {
    let score = 0;

    if (q1.value === "active") {
        score++;
    }

    if (q2.value === "active") {
        score++;
    }

    if (q3.value === "active") {
        score++;
    }

    resultBox.className = "result-box";

    if (score <= 1) {
        resultTitle.textContent = "のんびりタイプ";
        resultComment.textContent = "家でゆっくり過ごすと元気になれそうです。";
        resultBox.classList.add("relax");
    } else if (score === 2) {
        resultTitle.textContent = "バランスタイプ";
        resultComment.textContent = "休む時間と遊ぶ時間のバランスを取るのが得意です。";
        resultBox.classList.add("balance");
    } else {
        resultTitle.textContent = "アクティブタイプ";
        resultComment.textContent = "外に出て動くことでリフレッシュできそうです。";
        resultBox.classList.add("active");
    }
});
```

<Section title="JavaScriptで点数を計算する" goal="3つの質問の値を見て、active を選んだ数だけ score を増やします。">

`score` という変数を用意して、`q1.value` / `q2.value` / `q3.value` が `"active"` かどうかを `if` で判定します。

<Action title="JavaScriptを書く">

次のJavaScriptを追加します。

```js
const q1 = document.querySelector("#q1");
const q2 = document.querySelector("#q2");
const q3 = document.querySelector("#q3");

const button = document.querySelector("#diagnoseButton");
const resultBox = document.querySelector("#resultBox");
const resultTitle = document.querySelector("#resultTitle");
const resultComment = document.querySelector("#resultComment");

button.addEventListener("click", function () {
    let score = 0;

    if (q1.value === "active") {
        score++;
    }

    if (q2.value === "active") {
        score++;
    }

    if (q3.value === "active") {
        score++;
    }
});
```

</Action>

<Verify title="動きを確認する">

「診断する」ボタンを押しても結果は変わりませんが、`console.log(score);` を一時的に追加すると、active を選んだ数（0〜3）が表示されることを確認できます。

</Verify>

<Recovery title="button が反応しないとき">

`addEventListener` の `"click"` がクォートで囲まれているか、`#diagnoseButton` がHTML側の `id` と一致しているかを確認してください。

</Recovery>

</Section>

<Section title="点数で結果タイトルと色を切り替える" goal="score の値に応じて、結果タイトル・コメント・背景クラスを変えます。">

`resultBox.className = "result-box";` で前のクラスを消してから、`classList.add(...)` で新しいクラスを足すのがコツです。

<Concept title="className と classList の使い分け">

`className` に文字列を代入すると、クラスが**丸ごと置き換わります**。前に付いていたクラスを消したいときに使います。`classList.add(...)` はクラスを**1つだけ追加**するので、ベースとなる `result-box` を残したままタイプ別の色クラスを足す用途に向きます。

</Concept>

<Action title="判定処理を追加する">

`button.addEventListener` の中の末尾に、`if / else if / else` の判定を追加します。

```js
    resultBox.className = "result-box";

    if (score <= 1) {
        resultTitle.textContent = "のんびりタイプ";
        resultComment.textContent = "家でゆっくり過ごすと元気になれそうです。";
        resultBox.classList.add("relax");
    } else if (score === 2) {
        resultTitle.textContent = "バランスタイプ";
        resultComment.textContent = "休む時間と遊ぶ時間のバランスを取るのが得意です。";
        resultBox.classList.add("balance");
    } else {
        resultTitle.textContent = "アクティブタイプ";
        resultComment.textContent = "外に出て動くことでリフレッシュできそうです。";
        resultBox.classList.add("active");
    }
```

</Action>

<Verify title="結果を確認する">

3つの質問で「早起き／外で活動／すぐ行きたい」を全部選ぶと active が3つになり、結果が青系の「アクティブタイプ」に切り替わります。「ゆっくり寝たい／家でのんびり／予定が合えば行く」を全部選ぶと0〜1になり、緑系の「のんびりタイプ」になります。

</Verify>

<Recovery title="結果ボックスの色が変わらないとき">

`className = "result-box";` があるか確認してください。これを書き忘れると、2回目以降の診断でクラスが積み重なってしまいます。

</Recovery>

</Section>

<Section title="応用：テーマを変えて自分の診断にする" goal="質問・選択肢・結果・色を書き換えて、自分だけの診断アプリに改造します。">

テーマを変えるときは、`value` の名前と `if` の条件をセットで書き換えるのがポイントです。`active` / `relax` の名前を残したまま内容を入れ替えるより、テーマに合わせて `value` 自体を変えた方が `if` 文が読みやすくなります。

まずは演習1に取り組み、早く終わったら演習-発展に進みます。

### 演習-発展1

<Exercise>

質問を4つに増やします。質問が増えると `score` は最大4になるので、結果を分けるしきい値も見直します。次の手順で進めます。

1. HTMLに4つ目の質問セクションを追加する。
2. JavaScriptで `q4` を取得し、`active` 判定の `if` を1つ足す。
3. `score` が 0〜4 になるので、3タイプに分かれるよう条件を調整する。

確認するときは、4問すべてで「外で活動する」側を選ぶと `score` が4になり、アクティブタイプになることを見ます。

<Solution>

3つ目の質問の下に、4つ目の質問を追加します。

```html
<section class="question">
    <label for="q4">Q4. 連休があったら？</label>
    <select id="q4">
        <option value="relax">家でゆっくりしたい</option>
        <option value="active">どこかへ出かけたい</option>
    </select>
</section>
```

`q4` を取得し、判定を1つ足してから、しきい値を調整します。

```js
const q4 = document.querySelector("#q4");

button.addEventListener("click", function () {
    let score = 0;

    if (q1.value === "active") {
        score++;
    }

    if (q2.value === "active") {
        score++;
    }

    if (q3.value === "active") {
        score++;
    }

    if (q4.value === "active") {
        score++;
    }

    resultBox.className = "result-box";

    if (score <= 1) {
        resultTitle.textContent = "のんびりタイプ";
        resultComment.textContent = "家でゆっくり過ごすと元気になれそうです。";
        resultBox.classList.add("relax");
    } else if (score <= 3) {
        resultTitle.textContent = "バランスタイプ";
        resultComment.textContent = "休む時間と遊ぶ時間のバランスを取るのが得意です。";
        resultBox.classList.add("balance");
    } else {
        resultTitle.textContent = "アクティブタイプ";
        resultComment.textContent = "外に出て動くことでリフレッシュできそうです。";
        resultBox.classList.add("active");
    }
});
```

</Solution>

</Exercise>

### 演習-発展2

<Exercise>

最初は結果ボックスを隠しておき、「診断する」ボタンを押したときだけ表示します。次の手順で進めます。

1. CSSの `.result-box` に `display: none;` を足して、最初は隠す。
2. 判定処理の中で、結果ボックスを表示する1行を足す。

確認するときは、ページを開いた直後は結果ボックスが見えず、ボタンを押すと現れることを見ます。

<Solution>

CSSの `.result-box` に `display: none;` を追加します。

```css
.result-box {
    margin-top: 24px;
    padding: 20px;
    border-radius: 16px;
    background: #eeeeee;
    display: none;
}
```

判定処理の最後に、結果ボックスを表示する行を足します。

```js
    resultBox.style.display = "block";
```

`style.display` に `"block"` を入れると、CSSで隠していた要素を表示できます。

</Solution>

</Exercise>
