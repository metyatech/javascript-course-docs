問題データを配列で管理し、4つの選択肢ボタンから正解を選ぶクイズアプリを作ります。配列・オブジェクト・関数・イベントを組み合わせて、状態（いま何問目か・正解したか）を管理する流れを学びます。

<Section title="完成形を確認する" goal="完成したアプリの動作と、全体のおおまかな流れを先に確認します。">

完成すると、4つの選択肢から1つを選んで「次の問題へ」で進むクイズになります。全問終わると結果が表示され、選択肢はクリックできない状態になります。

| 操作 | 画面の変化 |
| --- | --- |
| 選択肢ボタンをクリックする | 正解・不正解が表示され、選択肢がクリックできなくなる |
| 「次の問題へ」ボタンを押す | 次の問題文と選択肢が表示される |
| 最後の問題を終える | 結果コメントが表示される |

```js
const questions = [
    {
        text: "JavaScriptでクリックイベントを登録するメソッドは？",
        choices: ["querySelector", "addEventListener", "textContent", "console.log"],
        answer: "addEventListener"
    },
    {
        text: "配列の最初の要素を取り出す番号は？",
        choices: ["0", "1", "2", "3"],
        answer: "0"
    },
    {
        text: "要素の文字を変更するプロパティは？",
        choices: ["value", "length", "textContent", "styleSheet"],
        answer: "textContent"
    }
];

let currentIndex = 0;
let score = 0;
let answered = false;

const progressText = document.querySelector("#progressText");
const questionText = document.querySelector("#questionText");
const messageText = document.querySelector("#messageText");
const nextButton = document.querySelector("#nextButton");

const choice1 = document.querySelector("#choice1");
const choice2 = document.querySelector("#choice2");
const choice3 = document.querySelector("#choice3");
const choice4 = document.querySelector("#choice4");

const choiceButtons = [choice1, choice2, choice3, choice4];

function showQuestion() {
    const question = questions[currentIndex];

    progressText.textContent = currentIndex + 1 + "問目 / " + questions.length + "問中";
    questionText.textContent = question.text;

    for (let i = 0; i < choiceButtons.length; i++) {
        choiceButtons[i].textContent = question.choices[i];
        choiceButtons[i].disabled = false;
    }

    messageText.textContent = "選択肢を選んでください。";
    messageText.className = "";
    answered = false;
}

function checkAnswer(selectedAnswer) {
    if (answered === true) {
        return;
    }

    const question = questions[currentIndex];

    if (selectedAnswer === question.answer) {
        score++;
        messageText.textContent = "正解！";
        messageText.className = "correct";
    } else {
        messageText.textContent = "不正解。正解は「" + question.answer + "」です。";
        messageText.className = "wrong";
    }

    answered = true;

    for (let i = 0; i < choiceButtons.length; i++) {
        choiceButtons[i].disabled = true;
    }
}

function showFinalResult() {
    progressText.textContent = "終了";
    questionText.textContent = "結果は " + questions.length + "問中 " + score + "問正解です。";
    messageText.textContent = "おつかれさまでした。";

    for (let i = 0; i < choiceButtons.length; i++) {
        choiceButtons[i].textContent = "-";
        choiceButtons[i].disabled = true;
    }

    nextButton.disabled = true;
}

choice1.addEventListener("click", function () {
    checkAnswer(choice1.textContent);
});

choice2.addEventListener("click", function () {
    checkAnswer(choice2.textContent);
});

choice3.addEventListener("click", function () {
    checkAnswer(choice3.textContent);
});

choice4.addEventListener("click", function () {
    checkAnswer(choice4.textContent);
});

nextButton.addEventListener("click", function () {
    if (answered === false) {
        messageText.textContent = "先に選択肢を選んでください。";
        return;
    }

    currentIndex++;

    if (currentIndex < questions.length) {
        showQuestion();
    } else {
        showFinalResult();
    }
});

showQuestion();
```

<Section title="問題データを配列で用意する" goal="問題文・選択肢・正解を、扱いやすいデータのかたちにまとめます。">

1問 = 1オブジェクトの形にし、3問を配列 `questions` に入れます。`answer` には `choices` の中の**どれかの文字列そのまま**を入れるのがルールです。

<Concept title="answer は choices のどれかと一致させる">

`answer` を「答えはB」のような説明文にしてしまうと、選んだ文字列と一致しなくなります。判定は `===` の単純比較なので、`choices` のどれかと一字一句同じにしておく必要があります。

</Concept>

<Section title="問題を表示する関数を作る" goal="現在の問題番号に対応する問題文と選択肢を画面に出します。">

`showQuestion` 関数は「問題を出すとき」にいつでも呼べるよう、副作用をまとめておく場所です。`for` 文で4つの選択肢ボタンを順に書き換えます。

<Concept title="for 文でボタンをまとめて更新する">

4つのボタンを1つずつ `choiceButtons[i].textContent = ...` で書き換えると、長くなるので `for` 文でまとめます。`i < choiceButtons.length`（`<=` ではなく `<`）が境界条件で、これを超えると `undefined` アクセスで止まります。

</Concept>

<Action title="showQuestion を書く">

```js
function showQuestion() {
    const question = questions[currentIndex];

    progressText.textContent = currentIndex + 1 + "問目 / " + questions.length + "問中";
    questionText.textContent = question.text;

    for (let i = 0; i < choiceButtons.length; i++) {
        choiceButtons[i].textContent = question.choices[i];
        choiceButtons[i].disabled = false;
    }

    messageText.textContent = "選択肢を選んでください。";
    messageText.className = "";
    answered = false;
}
```

最後に `showQuestion();` を1回呼んで、最初の問題を出します。

</Action>

<Verify title="表示を確認する">

ページを開いたとき「1問目 / 3問中」とJavaScriptクイズの1問目が表示され、4つの選択肢ボタンに `querySelector` `addEventListener` `textContent` `console.log` が入っていれば成功です。

</Verify>

<Recovery title="question.choices[i] が undefined になるとき">

`i <= choiceButtons.length` になっていないか確認してください。`<=` だと `i === 4` のとき配列範囲外になります。`i < choiceButtons.length`（`<`）が正解です。

</Recovery>

</Section>

<Verify title="判定を確認する">

選択肢ボタンをクリックして、正解なら緑系のメッセージ「正解！」、不正解なら赤系の「不正解。正解は〜」が表示されれば成功です。2回同じボタンを押しても score が増えないことも確認します。

</Verify>

</Recovery>

### 演習4（レベル4：バグ修正チャレンジ）

<Exercise>

次のコードは、わざと6か所にバグが入っています。`id` 名・変数名・比較演算子・配列の範囲・関数の呼び出しを観点にして探してください。

```js
const questions = [
    {
        text: "JavaScriptでクリックイベントを登録するメソッドは？",
        choices: ["querySelector", "addEventListener", "textContent", "console.log"],
        answer: "addEventListener"
    },
    {
        text: "配列の最初の要素を取り出す番号は？",
        choices: ["0", "1", "2", "3"],
        answer: "0"
    }
];

let currentIndex = 1;
let score = 0;
let answered = false;

const progressText = document.querySelector("#progress");
const questionText = document.querySelector("#questionText");
const messageText = document.querySelector("#messageText");
const nextButton = document.querySelector("#nextButton");

const choice1 = document.querySelector("#choice1");
const choice2 = document.querySelector("#choice2");
const choice3 = document.querySelector("#choice3");
const choice4 = document.querySelector("#choice4");

const choiceButtons = [choice1, choice2, choice3, choice4];

function showQuestion() {
    const question = questions[currentIndex];

    progressText.textContent = currentIndex + 1 + "問目 / " + questions.length + "問中";
    questionText.textContent = question.text;

    for (let i = 0; i <= choiceButtons.length; i++) {
        choiceButtons[i].textContent = question.choices[i];
        choiceButtons[i].disabled = false;
    }

    messageText.textContent = "選択肢を選んでください。";
    answered = false;
}

function checkAnswer(selectedAnswer) {
    if (answered = true) {
        return;
    }

    const question = questions[currentIndex];

    if (selectedAnswer === question.answer) {
        score++;
        messageText.textContent = "正解！";
    } else {
        messageText.textContent = "不正解。正解は「" + question.answer + "」です。";
    }

    answered = true;
}

choice1.addEventListener("click", function () {
    checkAnswer(choice1.textContent);
});

choice2.addEventListener("click", function () {
    checkAnswer(choice2.textContent);
});

choice3.addEventListener("click", function () {
    checkAnswer(choice3.textContent);
});

choice4.addEventListener("click", function () {
    checkAnswer(choice4.textContent);
});

nextButton.addEventListener("click", function () {
    currentIndex++;

    if (currentIndex < questions.length) {
        showQuestion;
    }
});

showQuestion();
```

<Solution>

1. `currentIndex = 1` になっている → `0` にする（最初の問題から始めるため）。
2. `#progress` になっている → HTML側が `#progressText` なので `#progressText` に合わせる。
3. `i <= choiceButtons.length` になっている → `i < choiceButtons.length` にする（配列範囲外アクセスを防ぐ）。
4. `if (answered = true)` になっている → `if (answered === true)` にする（代入と比較の混同を修正）。
5. `showQuestion;` になっている → `showQuestion();` にする（関数を実際に呼び出す）。
6. 最後の問題まで進んだあとに `showFinalResult()` を呼ぶ処理がない → `nextButton` のハンドラに「`else` なら `showFinalResult()` を呼ぶ」を追加する。

</Solution>

</Exercise>
