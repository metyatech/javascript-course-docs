ボタンを押すと結果が変わる、小さなランダムアプリを作ります。

このページでは、最初から完成コードを写すのではなく、次の順番で少しずつアプリにしていきます。

1. 画面のどこを変えるのかを見つける
2. ボタンで固定の文字を表示する
3. 複数の候補をデータとしてまとめる
4. ランダムに1つ選んで表示する
5. 自分のテーマに作り変える

---

<Section title="HTMLで変更される場所を用意する" goal="JavaScriptから変更したい場所にidを付けます。">

JavaScriptは、HTMLの部品を直接探して、文字を変更します。  
そのため、あとで変更したい場所には `id` を付けておきます。

<Concept title="id は JavaScript の目印">

`id="fortune"` のように名前を付けると、JavaScriptでは `document.querySelector('#fortune')` でその場所を探せます。

`#` は「idを探す」という意味です。  
HTML側の名前とJavaScript側の名前が1文字でも違うと、正しく探せません。

</Concept>

<Action title="HTMLを書く">

`<body>` の中に、次のHTMLを書きます。

```html
<main class="app">
    <h1>今日の運勢メーカー</h1>
    <p class="lead">ボタンを押して、今日の運勢を占いましょう。</p>

    <div class="card">
        <p id="fortune" class="fortune">まだ占っていません</p>
        <p id="message" class="message">下のボタンを押してください。</p>
    </div>

    <button id="drawButton">占う</button>
</main>
```

</Action>

<Action title="CSSとJavaScriptを読み込む">

`index.html` の `<head>` の中に、CSSとJavaScriptの読み込みを書きます。`script` に `defer` を付けると、HTMLを読み込み終えてからJavaScriptが実行されます。

```html
<head>
    <link rel="stylesheet" href="style.css">
    <script src="script.js" defer></script>
</head>
```

`style.css` と `script.js` は、このあとのステップで作ります。

</Action>

<Section title="ボタンで固定の文字を表示する" goal="まずはランダムにせず、クリックで必ず同じ結果を表示します。">

最初からランダムにすると、何が起きているのか分かりにくくなります。  
まずは、ボタンを押したら必ず「大吉」と表示されるところまで作ります。

<Exercise>

次のコードを実行すると、どの部分の文字が変わるでしょうか。

```js
const fortune = document.querySelector('#fortune');
fortune.textContent = '大吉';
```

<Solution>

`id="fortune"` の場所の文字が `大吉` に変わります。

`fortune` は、JavaScriptで取得したHTML要素です。  
`textContent` に文字列を代入すると、その要素の表示文字が変わります。

</Solution>

</Exercise>

<Action title="要素を取得してクリックイベントを書く">

`script.js` に次のJavaScriptを書きます。

```js
const fortune = document.querySelector('#fortune');
const message = document.querySelector('#message');
const drawButton = document.querySelector('#drawButton');

drawButton.addEventListener('click', () => {
    fortune.textContent = '大吉';
    message.textContent = '今日は良いことがありそうです。';
});
```

</Action>

<Verify title="動きを確認する">

「占う」ボタンを押して、運勢が「大吉」、メッセージが「今日は良いことがありそうです。」に変われば成功です。

</Verify>

<Recovery title="ボタンを押しても変わらないとき">

次の3つを確認します。

1. HTMLに `id="drawButton"` があるか
2. JavaScriptで `document.querySelector('#drawButton')` と書いているか
3. `addEventListener('click', ...)` の `click` がクォートで囲まれているか

</Recovery>

</Section>

<Section title="候補をデータとしてまとめる" goal="運勢とメッセージを、配列とオブジェクトでまとめます。">

今のままだと、結果は1種類だけです。  
複数の結果を扱うために、運勢とメッセージをデータとしてまとめます。

<Concept title="1つの結果をオブジェクトにする">

1つの運勢には、名前とメッセージがあります。

```js
{
    name: '大吉',
    text: '新しいことに挑戦すると、よい発見がありそうです。'
}
```

`name` は運勢の名前、`text` はメッセージです。  
このように、関係する情報を1つにまとめたものをオブジェクトとして扱います。

</Concept>

<Action title="results配列を書く">

`script.js` の上の方に、次の配列を追加します。

```js
const results = [
    {
        name: '大吉',
        text: '新しいことに挑戦すると、よい発見がありそうです。'
    },
    {
        name: '中吉',
        text: '落ち着いて進めると、予定どおりに進みそうです。'
    },
    {
        name: '小吉',
        text: '小さな確認を大切にすると、ミスを防げそうです。'
    }
];
```

</Action>

<Exercise>

次のコードで取り出されるのは、どの運勢でしょうか。

```js
const result = results[1];
```

<Solution>

`results[1]` で取り出されるのは、2番目のデータです。  
配列の番号は `0` から始まるため、`results[0]` が大吉、`results[1]` が中吉です。

</Solution>

</Exercise>

<Action title="まずは2番目の結果を表示する">

クリックイベントの中を、次のように書き換えます。

```js
drawButton.addEventListener('click', () => {
    const result = results[1];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

</Action>

<Verify title="配列から表示できるか確認する">

ボタンを押して「中吉」とそのメッセージが表示されれば成功です。  
ここではまだランダムではありません。

</Verify>

</Section>

<Section title="ランダムに1つ選ぶ" goal="配列の長さを使って、候補の中から1つ選びます。">

ランダムにするには、配列の番号をランダムに作ります。

<Concept title="ランダムな配列番号を作る">

`Math.random()` は、0以上1未満の小数を作ります。

3個の候補から選びたい場合は、次のようにします。

```js
const index = Math.floor(Math.random() * results.length);
```

`results.length` は候補の数です。  
候補が3個なら、`index` は `0`、`1`、`2` のどれかになります。

</Concept>

<Exercise>

`results.length` が5のとき、次のコードで作られる `index` はどの範囲になりますか。

```js
const index = Math.floor(Math.random() * results.length);
```

<Solution>

`0`、`1`、`2`、`3`、`4` のどれかになります。  
配列の最後の番号は、長さより1小さい番号です。

</Solution>

</Exercise>

<Action title="ランダム表示にする">

クリックイベントを次のように書き換えます。

```js
drawButton.addEventListener('click', () => {
    const index = Math.floor(Math.random() * results.length);
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

</Action>

<Verify title="ランダムに変わるか確認する">

「占う」ボタンを何度も押して、大吉・中吉・小吉のいずれかが表示されれば成功です。  
同じ結果が続くこともあります。ランダムなので、それはバグではありません。

</Verify>

</Section>

<Section title="バグ修正チャレンジ" goal="よくあるミスを見つけて、自分で直せるようにします。">

次のコードには、よくあるミスが4つ入っています。  
どこが間違っているか探してください。

<Exercise>

```js
const fortune = document.querySelector('fortune');
const message = document.querySelector('#message');
const drawButton = document.querySelector('#drawbutton');

const results = [
    {
        name: '大吉',
        text: '新しいことに挑戦すると、よい発見がありそうです。'
    },
    {
        name: '中吉',
        text: '落ち着いて進めると、予定どおりに進みそうです。'
    }
];

drawButton.addEventListener('click', () => {
    const index = Math.floor(Math.random() * result.length);
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

<Solution>

1. `document.querySelector('fortune')` は `#` がないため、`document.querySelector('#fortune')` に直す。
2. `#drawbutton` はHTMLの `id="drawButton"` と大文字小文字が違うため、`#drawButton` に直す。
3. `result.length` は存在しない変数なので、`results.length` に直す。
4. `const result = results[index];` より前に `result.length` を使おうとしているため、候補数は `results.length` から取る。

修正版は次の形です。

```js
const fortune = document.querySelector('#fortune');
const message = document.querySelector('#message');
const drawButton = document.querySelector('#drawButton');

const results = [
    {
        name: '大吉',
        text: '新しいことに挑戦すると、よい発見がありそうです。'
    },
    {
        name: '中吉',
        text: '落ち着いて進めると、予定どおりに進みそうです。'
    }
];

drawButton.addEventListener('click', () => {
    const index = Math.floor(Math.random() * results.length);
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

</Solution>

</Exercise>

</Section>
