ボタンをクリックすると今日の運勢が表示される、小さな占いアプリを作ります。HTMLで部品を用意し、CSSで見た目を整え、JavaScriptでクリック時の変化を作ります。

<Section title="完成形を確認する" goal="作るアプリの動きと、使う部品を先に確認します。">

完成すると、ボタンをクリックするたびに運勢とメッセージが変わる画面になります。

| 操作 | 画面の変化 |
| --- | --- |
| 「占う」ボタンをクリックする | 大吉・中吉・小吉のどれかと、短いメッセージが表示される |
| もう一度クリックする | 別の結果に変わることがある |

<Action title="HTMLを書く">

`<body>` の中に、次のHTMLを用意します。

```html
<main class="app">
    <h1>今日の運勢メーカー</h1>
    <p class="lead">ボタンを押して、今日の運勢を占いましょう。</p>

    <div class="card">
        <p id="fortune" class="fortune">まだ占っていません</p>
        <p id="message" class="message">下のボタンを押してください。</p>
    </div>

    <button id="drawButton">占う</button>
</main>
```

</Action>

<Verify title="部品を確認する">

画面に見出し、白いカード、占うボタンが表示されていれば成功です。この時点では、ボタンを押してもまだ何も起きません。

</Verify>

<Section title="JavaScriptで結果を変える" goal="クリックしたときに、運勢とメッセージを書き換えます。">

`results` は、運勢の候補をまとめた配列です。クリックされたら、`Math.random()` で候補の番号を選び、`textContent` で画面の文字を入れ替えます。

<Concept title="Math.random() の役割">

`Math.random()` は、0以上1未満のランダムな数を作ります。配列の番号として使うため、`results.length` をかけて候補数に合わせ、`Math.floor()` で整数にします。

</Concept>

<Action title="JavaScriptを書く">

次のJavaScriptを追加します。

```js
const fortune = document.querySelector('#fortune');
const message = document.querySelector('#message');
const drawButton = document.querySelector('#drawButton');

const results = [
    {
        name: '大吉',
        text: '新しいことに挑戦すると、よい発見がありそうです。'
    },
    {
        name: '中吉',
        text: '落ち着いて進めると、予定どおりに進みそうです。'
    },
    {
        name: '小吉',
        text: '小さな確認を大切にすると、ミスを防げそうです。'
    }
];

drawButton.addEventListener('click', () => {
    const index = Math.floor(Math.random() * results.length);
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

</Action>

<Verify title="動きを確認する">

「占う」ボタンを押して、運勢とメッセージが変われば成功です。何度か押して、結果が変わることも確認してください。

</Verify>

<Recovery title="ボタンを押しても変わらないとき">

`id` の名前を確認してください。HTMLの `id="drawButton"` と、JavaScriptの `document.querySelector('#drawButton')` は同じ名前にする必要があります。

</Recovery>

</Section>

<Section title="応用：自分の運勢メーカーに作り変える" goal="候補を増やし、表示や演出を足して、自分だけの占いに広げます。">

結果を増やすときは、`results` 配列に同じ形のデータを追加します。配列の長さは `results.length` で自動的に数えられるため、ランダムに選ぶ処理は変更しなくてかまいません。

まずは演習1に取り組み、早く終わったら演習-発展に進みます。

### 演習-発展1

<Exercise>

占い結果に「ラッキーアイテム」を一緒に表示します。次の手順で進めます。

1. カードの中に、アイテムを表示する段落を1つ追加する。
2. `results` の各オブジェクトに `item` プロパティを足す。
3. クリックしたとき、その `item` を画面に表示する。

確認するときは、「占う」ボタンを押すたびに、運勢・メッセージに加えてラッキーアイテムも変わることを見ます。

<Solution>

カードの中に、アイテムの表示先を追加します。

```html
<div class="card">
    <p id="fortune" class="fortune">まだ占っていません</p>
    <p id="message" class="message">下のボタンを押してください。</p>
    <p id="item" class="message">ラッキーアイテム：－</p>
</div>
```

`results` の各オブジェクトに `item` を足し、クリック時に表示します。

```js
const item = document.querySelector('#item');

const results = [
    {
        name: '大吉',
        text: '新しいことに挑戦すると、よい発見がありそうです。',
        item: '青いペン'
    },
    {
        name: '中吉',
        text: '落ち着いて進めると、予定どおりに進みそうです。',
        item: '温かい飲み物'
    },
    {
        name: '小吉',
        text: '小さな確認を大切にすると、ミスを防げそうです。',
        item: 'メモ帳'
    }
];

drawButton.addEventListener('click', () => {
    const index = Math.floor(Math.random() * results.length);
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
    item.textContent = 'ラッキーアイテム：' + result.item;
});
```

</Solution>

</Exercise>

### 演習-発展2

<Exercise>

「占う」ボタンを連続で押したとき、直前と同じ運勢が続けて出ないようにします。

ヒント：直前に表示した番号を覚えておく変数を用意し、同じ番号が出たら選び直します。

確認するときは、何度も押して、2回続けて同じ運勢が出ないことを見ます。

<Solution>

```js
let lastIndex = -1;

drawButton.addEventListener('click', () => {
    let index = Math.floor(Math.random() * results.length);

    while (index === lastIndex) {
        index = Math.floor(Math.random() * results.length);
    }

    lastIndex = index;
    const result = results[index];

    fortune.textContent = result.name;
    message.textContent = result.text;
});
```

`lastIndex` に前回の番号を覚えておき、同じ番号が出たら `while` で選び直します。

</Solution>

</Exercise>

</Section>
