<Exercise title="演習3">

「都道府県を入力して下さい」という内容の `prompt` と、「市区町村を入力して下さい」という内容の `prompt` を出し、それらを合体させて「都道府県 市区町村」の形式で出力せよ。

例）都道府県に「埼玉県」、市区町村に「さいたま市」と入力した場合：`埼玉県 さいたま市`

<Solution>
<CodePreview>
```javascript
let todofuken = prompt("都道府県を入力して下さい");
let shikuchoson = prompt("市区町村を入力して下さい");

console.log(todofuken + " " + shikuchoson);
```
</CodePreview>
</Solution>

</Exercise>

<Exercise title="演習4">

次の各変数の中身と、その型を答えよ。

```js
let A = 1.2 + 2;
let B = "天気は" + "晴れ";
let C = 15 + "11";
```

<Solution>

- `A` の値：`3.2`　型：**数値型（Number）**
- `B` の値：`"天気は晴れ"`　型：**文字列型（String）**
- `C` の値：`"1511"`　型：**文字列型（String）**　← 数値 + 文字列 は文字列になる

</Solution>

</Exercise>

<Exercise title="演習9">

次の関数 `add` を使い、`add(2)` を呼び出して返ってきた値を変数 `kekka` に入れ、出力するコードを書け。

```js
function add(x) {
    let result = x + 1;
    return result;
}
```

<Solution>
<CodePreview>
```javascript
function add(x) {
    let result = x + 1;
    return result;
}

let kekka = add(2);
console.log(kekka); // 出力: 3
```
</CodePreview>
</Solution>

</Exercise>

<Exercise title="演習10">

for 文を使って、「3」「4」「5」「6」「7」「8」「9」「10」をこの順でそれぞれ出力せよ。

<Solution>
<CodePreview>

```javascript
    console.log(i);
}
```
</CodePreview>
</Solution>

<Exercise title="演習11">

for 文を使って、10〜100 の整数の和を求めて出力せよ。

<Solution>
<CodePreview>
```javascript
let sum = 0;

for (let i = 10; i <= 100; i++) {
    sum = sum + i;
}

console.log(sum); // 出力: 5005
```
</CodePreview>
</Solution>

</Exercise>
