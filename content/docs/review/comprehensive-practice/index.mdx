## 型の確認

**型**とは値の種類のこと。`"あいうえお"` → 文字列型、`123`・`11.7` → 数値型。

次の各変数の中身と、その型を確認してみよう。

```js
let A = 1.2 + 2;        // 数値 + 数値 → 数値型  値: 3.2
let B = "天気は" + "晴れ"; // 文字列 + 文字列 → 文字列型  値: "天気は晴れ"
let C = 15 + "11";      // 数値 + 文字列 → 文字列型  値: "1511"
```

## 関数（function）

**関数**とは、何か入力を与えると、処理を行って、何かを結果として出すものです。

```js
function 関数名(入力を受け取る名前) {
    処理
    return 結果;
}
```

例）入力を `x` という名前の変数で受け取って、`x + 1` を結果とする関数 `add`：

```js
function add(x) {
    let result = x + 1;
    return result;
}
```

作った関数は、`関数名(入力に渡す値)` の形で**呼び出し**て使います。

```js
// add(2) の部分が関数呼び出し
let kekka = add(2);
console.log(kekka); // 出力: 3
```

<Exercise title="演習9">
