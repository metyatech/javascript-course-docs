## 条件分岐とは？

条件によって実行する処理を切り替えることを**条件分岐**といいます。
「点数が80点以上なら合格メッセージを表示する」「入力値によって返答を変える」といった場面で使います。

---

## if 文

条件が **真（true）** のときだけ、波括弧 `{}` 内の処理を実行します。

```js
if (条件) {
    // 条件が true のときの処理
}
```

<CodePreview>
```javascript
let answer = prompt('日本一高い山は？');

if (answer === '富士山') {
    console.log('正解！'); // answer が "富士山" のときだけ出力される
}
```
</CodePreview>

## JavaScriptのインデント

JavaScriptでは、**波括弧 `{ }` で囲まれたブロックの中**をインデントします。
インデントを適切に設定しないと、どこからどこまでが一つのまとまり（ブロック）なのかが分からなくなり、エラーの原因になります。

:::note
インデントの基本的な考え方は、次の共通基礎で確認できます。

- [入れ子とインデント](https://web-foundations-docs.vercel.app/docs/nesting-indentation)
:::

### 良い例（インデントが揃っている）

```javascript
if (条件) {
    // 波括弧の中なのでインデントを1段下げる
    console.log('真です');
}
```

### 悪い例（インデントがぐちゃぐちゃ）

{/*prettier-ignore*/}

```javascript
if (条件) {
console.log("真です");
}
```

悪い例では、コードがどの波括弧に含まれているのかが視覚的に分かりづらくなっています。プロの現場ではインデントが揃っていないコードはそれだけで「読みにくい」と評価されてしまいます。VSCodeではコードを選択して `Alt + Shift + F` (Windows) または `Option + Shift + F` (Mac) で自動的に綺麗に揃える機能（フォーマット）もありますが、まずは自分で「波括弧の次は下げる」意識を持ちましょう。

### 演習: インデントを直そう

<Exercise>

以下のJavaScriptコードは、インデントが全くされておらず非常に読みにくい状態です。正しいインデント（波括弧 `{ }` の中を1段下げる）に修正してください。

{/*prettier-ignore*/}

<CodePreview>
```javascript
let score = 85;
if (score >= 80) {
console.log("合格です！");
console.log("よくがんばりました。");
}
```
</CodePreview>

<Solution>

<CodePreview>
```javascript

if (score >= 80) {
    console.log('合格です！');
    console.log('よくがんばりました。');
}
```
</CodePreview>

</Solution>

</Exercise>

## if...else 文

条件が **真のとき** と **偽のとき** で、それぞれ別の処理を実行できます。

```js
if (条件) {
    // true のときの処理
} else {
    // false のときの処理
}
```

<CodePreview>
```javascript
let input = prompt('helloを入力して下さい');

if (input === 'hello') {
    console.log('helloだね'); // "hello" を入力したとき
} else {
    console.log('helloじゃないよ'); // それ以外のとき
}
```
</CodePreview>

## else if 文

「〇〇のとき」→「そうじゃなくて△△のとき」→「それ以外のとき」のように、**3つ以上の分岐**を作りたい場合は `else if` を使います。

```js
if (条件1) {
    // 条件1 が true のときの処理
} else if (条件2) {
    // 条件1 が false かつ 条件2 が true のときの処理
} else {
    // どの条件も false のときの処理
}
```

<CodePreview>
```javascript
let number = Number(prompt('数値を入力して下さい'));

if (number >= 100) {
    console.log('100以上');
} else if (number >= 50) {
    console.log('50以上100未満');
} else {
    console.log('50未満');
}
```
</CodePreview>
