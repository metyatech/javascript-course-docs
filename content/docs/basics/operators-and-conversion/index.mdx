## 真偽値型（boolean）

`true`（真）または `false`（偽）の2つの値だけをとる型です。
次に紹介する演算子を使うと、比較結果が真偽値として得られます。

---

## 等価演算子

2つの値が等しいかどうかを調べます。

| 演算子 | 意味 | 例 |
| :-: | :-: | :-: |
| `===` | 等しい | `1 === 1` → `true` |
| `!==` | 等しくない | `1 !== 2` → `true` |

:::caution
`===` は値と**型**が両方一致するときだけ `true` になります。`1 === "1"` は `false` です（片方が数値、もう片方が文字列のため）。
:::

<CodePreview>
```javascript
console.log(1 === 1);        // 出力: true
console.log(1 === "1");      // 出力: false （型が違う）
console.log("abc" === "abc"); // 出力: true
console.log(1 !== 2);        // 出力: true
```
</CodePreview>

## 関係演算子

数値の大小を比べます。

| 演算子 | 意味 | 例 |
| :-: | :-: | :-: |
| `<` | より小さい | `1 < 2` → `true` |
| `>` | より大きい | `1 > 2` → `false` |
| `<=` | 以下 | `1 <= 1` → `true` |
| `>=` | 以上 | `2 >= 3` → `false` |

<CodePreview>
```javascript
console.log(5 < 10);  // 出力: true
console.log(7 > 7);   // 出力: false
console.log(8 <= 8);  // 出力: true
console.log(9 >= 10); // 出力: false
```
</CodePreview>

## 論理演算子

複数の条件を組み合わせるときに使います。

| 演算子 | 意味 | 説明 |
| :-: | :-: | :-: |
| `&&` | かつ（AND） | 両方 `true` のとき `true` |
| `\|\|` | または（OR） | どちらか一方でも `true` なら `true` |

<CodePreview>
```javascript
console.log(true && true);   // 出力: true
console.log(true && false);  // 出力: false
console.log(false || true);  // 出力: true
console.log(false || false); // 出力: false

// 組み合わせ例：5以上 かつ 10以下
let num = 7;
console.log(num >= 5 && num <= 10); // 出力: true
```
</CodePreview>

---

## 論理否定演算子

`!` を使うと、真偽値を反転できます。

| 式 | 結果 |
| :-: | :-: |
| `!true` | `false` |
| `!false` | `true` |

<CodePreview>
```javascript
console.log(!true);  // 出力: false
console.log(!false); // 出力: true

// 条件全体を反転する例
let value = 5;
console.log(!(value === 5)); // 出力: false （value === 5 は true → その逆）
```
</CodePreview>
