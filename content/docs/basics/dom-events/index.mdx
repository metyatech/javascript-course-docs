## JavaScript で要素を取得するには

HTML の解析が終わった後でないと、JavaScript から要素を取得できません。
外部 JS ファイルを読み込む `<script>` タグに `defer` をつけると、HTML の解析が終わった後に実行されます。

```html
<script src="index.js" defer></script>
```

要素を取得するには `document.querySelector` を使います。

| 関数 | 動作 |
| :-- | :-- |
| `document.querySelector(セレクタ)` | 指定した CSS のセレクタに一致する最初の要素を返す |

- ID 名で指定する場合は `"#ID名"` の形式で書きます。

<CodePreview>
```html
<input type="text" value="サンプルテキスト" id="abc">
```

```javascript
// id が "abc" の要素を取得する
let abcYoso = document.querySelector("#abc");

// 要素を出力してみる
console.log(abcYoso);

// .value で入力値を取得する
console.log(abcYoso.value);
```
</CodePreview>

:::caution
`.value` で取得した値は**文字列型**です。数値として計算するには `Number()` で変換してください。
:::

## イベント

「ボタンをクリックしたとき」のように「○○したとき」を表すものを**イベント**といいます。
`addEventListener` を使うと、○○が起きたときに××を行う、という処理を設定できます。

| 関数 | 動作 |
| :-- | :-- |
| `要素.addEventListener(タイプ, コールバック関数)` | 要素に対して指定したタイプのイベントが起きた時、コールバック関数を呼び出す |

代表的なイベントタイプ：

| タイプ | 意味 |
| :-: | :-: |
| `click` | クリックしたとき |
| `input` | 入力値を変更したとき |
| `keydown` | キーを押したとき |
| `keyup` | キーを離したとき |

<CodePreview>
```html
<input type="text" value="サンプルテキスト" id="abc">
```

```javascript
// テキストボックスの要素を取得
let abcYoso = document.querySelector("#abc");

// テキストボックスをクリックしたとき
abcYoso.addEventListener("click", function () {
    // これを行う
    console.log("テキストボックスの値は：" + abcYoso.value);
});
```
</CodePreview>

以下のような料金計算機を作れ。

- 大人は1人 500 円、子供は1人 300 円
- 「計算」ボタンを押すと、入力された人数から合計金額を計算して「合計金額」欄に表示する
- 例）大人 1 人・子供 2 人 → 1100 円

<CodePreview
  sourceId="演習1"
  htmlVisible={true}
  jsVisible={false}
  previewVisible={true}
  share={false}>
```html
<p>大人は1人500円、子供は1人300円です。</p>
<p>大人 <input type="number" id="adult" value="1"> 人</p>
<p>子供 <input type="number" id="child" value="2"> 人</p>
<button id="calcBtn">計算</button>
<p>合計金額 <input type="text" id="total" readonly> 円</p>
```

```javascript
// 要素を取得する
let adultYoso = document.querySelector("#adult");
let childYoso = document.querySelector("#child");
let calcBtnYoso = document.querySelector("#calcBtn");
let totalYoso = document.querySelector("#total");

// 計算ボタンをクリックしたとき
calcBtnYoso.addEventListener("click", function () {
    let adult = Number(adultYoso.value);
    let child = Number(childYoso.value);
    // 合計金額を計算して表示する
    totalYoso.value = adult * 500 + child * 300;
});
```
</CodePreview>
