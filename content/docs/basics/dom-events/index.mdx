## defer と document.querySelector

HTML 内の要素を JavaScript で操作するには、HTML の解析が終わった後に JavaScript を実行する必要があります。
外部 JS ファイルを読み込む `<script>` タグに `defer` をつけると、HTML の解析が終わった後に実行されます。

```html
<script src="index.js" defer></script>
```

要素を取得するには `document.querySelector` を使います。引数には CSS セレクタを指定します。
ID で指定する場合は `"#ID名"` の形式で書きます。

```js
// id が "abc" の要素を取得する
let abcYoso = document.querySelector("#abc");
```

## イベントとは

**イベント** とは、「ページの読込完了」、「ユーザーがウェブページ上で操作（クリック、マウス移動など）を行った」などの、何らかの「出来事」のことです。
JavaScriptを使うと、このような出来事が起きたときに反応して何か処理を実行することができます。

イベントに反応させるには、次のような書き方をします。

```js
要素.addEventListener("イベント名", function() {
  // イベントが起きたときに実行する処理
});
```

## click イベント

**click** イベントは、要素がクリックされたときに発生します。

```js
要素.addEventListener("click", function() {
  // クリックされたときの処理
});
```

<CodePreview>
```html
<button>クリックしてね</button>
<p>まだクリックされていません</p>
```

```javascript
// ボタンがクリックされたら、メッセージを変更する
let buttonYoso = document.querySelector("button");
let pYoso = document.querySelector("p");

buttonYoso.addEventListener("click", function() {
  pYoso.innerText = "ボタンがクリックされました！";
});
```
</CodePreview>

<CodePreview>
```html
<div></div>
```

```css
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
}
```

```javascript
// ボックスをクリックしたら、背景色を赤に変更
let divYoso = document.querySelector("div");

divYoso.addEventListener("click", function() {
  divYoso.style.backgroundColor = "red";
});
```
</CodePreview>

## フォーム入力値の取得（.value）

`input` 要素に入力された値は、`.value` プロパティで取得できます。

```js
let inputYoso = document.querySelector("input");
let nyuryokuChi = inputYoso.value; // 入力された値を取得
```

<CodePreview>
```html
<input type="text" placeholder="名前を入力">
<button>表示</button>
<p></p>
```

```javascript
// ボタンがクリックされたら入力値を取得して表示する
let inputYoso = document.querySelector("input");
let buttonYoso = document.querySelector("button");
let pYoso = document.querySelector("p");

buttonYoso.addEventListener("click", function() {
    // .value で入力されたテキストを取得する
    let nyuryokuChi = inputYoso.value;
    pYoso.innerText = "入力された値: " + nyuryokuChi;
});
```
</CodePreview>

:::caution
`.value` で取得した値は **文字列**（テキスト）です。数値として計算するには `Number()` で変換する必要があります。

```js
let inputYoso = document.querySelector("input");
let suuji = Number(inputYoso.value); // 文字列 → 数値に変換
```
:::

<Exercise title="演習6（料金計算）">
**body内が** 次のようになっているHTMLファイルに対し、JavaScriptで次の動作を実装せよ。

> - 「時間」の入力欄に数値を入力してボタンをクリックすると、料金（時間 × 500円）を計算して表示する
> - 表示例：「料金: 1500円」（3時間の場合）

<CodePreview
  sourceId="演習6"
  htmlVisible={true}
  jsVisible={false}
  previewVisible={true}>
```html
<label>時間: <input type="number" id="timeInput" placeholder="例: 3"></label>
<button id="calcBtn">計算</button>
<p id="resultText">料金: 0円</p>
```

```javascript
// 入力欄・ボタン・結果表示をそれぞれ取得する
let timeInputYoso = document.querySelector("#timeInput");
let calcBtnYoso = document.querySelector("#calcBtn");
let resultTextYoso = document.querySelector("#resultText");

calcBtnYoso.addEventListener("click", function() {
    // 入力値を数値に変換する
    let jikan = Number(timeInputYoso.value);
    // 料金を計算する（1時間あたり500円）
    let ryokin = jikan * 500;
    // 料金を表示する
    resultTextYoso.innerText = "料金: " + ryokin + "円";
});
```
</CodePreview>
