---

## マスターマインドって何？

マスターマインドは、0〜9の数字を2つ並べた2桁の正解を当てるゲームです。予想を入れると、コンピュータが「ヒット」と「ブロー」で何桁合っているかを教えてくれます。

- **ヒット**：数字も桁（十の位 / 一の位）も合っている個数
- **ブロー**：数字は合っているが、桁が違う個数

正解が `3` と `7`（つまり `37`）のときは、次のように判定します。

| 十の位の予想 | 一の位の予想 | 返事 | 意味 |
| :--: | :--: | :-- | :-- |
| 5 | 6 | `0ヒット 0ブロー` | どちらの数字も外れ |
| 3 | 6 | `1ヒット 0ブロー` | 十の位の `3` が一致 |
| 7 | 3 | `0ヒット 2ブロー` | 数字はどちらも合っているが、桁が逆 |
| 3 | 7 | `2ヒット 0ブロー` | クリアです |

### 演習1

<Exercise>

まずは、プレイヤーの予想を受け取って、入力チェックができるようにしましょう。正解は今だけ、十の位を `3`、一の位を `7` と固定しておきます。

`prompt` で「十の位」と「一の位」を別々に聞いて、どちらも `0`〜`9` なら `OK`、どちらかでも範囲外なら `invalid` と出力してください。テストしやすいように、今は正解も表示しておきます。

<Solution>
<CodePreview>
```javascript
    let secret0 = 3;
    let secret1 = 7;
    console.log('正解: ' + secret0 + secret1); // 出力: 正解: 37

    let guess0 = Number(prompt('十の位を入力してね（0〜9）'));
    let guess1 = Number(prompt('一の位を入力してね（0〜9）'));

    let valid0 = guess0 >= 0 && guess0 <= 9;
    let valid1 = guess1 >= 0 && guess1 <= 9;

    if (valid0 === true && valid1 === true) console.log('OK'); // 出力: OK
    else console.log('invalid'); // 出力: invalid
```
</CodePreview>
</Solution>
</Exercise>

### 演習3

<Exercise>

いよいよ、ヒットとブローを数えます。入力チェックと重複チェックの両方をクリアしたときだけ判定してください。

- ヒット：十の位どうし、または一の位どうしが同じ数字
- ブロー：十の位と一の位を入れ替えると数字が合う

判定結果は `ヒット: N、ブロー: M` の形で1行に出してください。ヒットが 2 なら続けて `クリア！` も出力します。

<Solution>
<CodePreview>
```javascript
    let secret0 = 3;
    let secret1 = 7;
    console.log('正解: ' + secret0 + secret1); // 出力: 正解: 37

    let guess0 = Number(prompt('十の位を入力してね（0〜9）'));
    let guess1 = Number(prompt('一の位を入力してね（0〜9）'));

    let valid0 = guess0 >= 0 && guess0 <= 9;
    let valid1 = guess1 >= 0 && guess1 <= 9;
    let duplicate = guess0 === guess1;

    if (valid0 === false || valid1 === false) console.log('invalid'); // 出力: invalid
    else if (duplicate === true) console.log('重複'); // 出力: 重複
    else {
        let hit = 0;
        let blow = 0;
        if (secret0 === guess0) hit = hit + 1;
        if (secret1 === guess1) hit = hit + 1;
        if (secret0 === guess1 && secret0 !== guess0) blow = blow + 1;
        if (secret1 === guess0 && secret1 !== guess1) blow = blow + 1;
        console.log('ヒット: ' + hit + '、ブロー: ' + blow); // 出力: ヒット: 0、ブロー: 2（37 を 73 と答えた場合）
        if (hit === 2) console.log('クリア！'); // 出力: クリア！
    }
```
</CodePreview>
</Solution>
</Exercise>

### 演習4

<Exercise>

最後に、5回まで予想できるゲームにします。演習3までの入力チェック・重複チェック・ヒット&ブロー判定を `for` で 5 回くり返し、ヒットが 2 になったら何回目で当たったかを保存して `break` してください。ループの後で、成功か失敗かに対応したメッセージを出します。

また、テスト用に入れていた `console.log('正解: ' + secret0 + secret1);` はコメントアウトして、プレイヤーには答えが見えないようにしてください。

<Solution>
<CodePreview>
```javascript
    let secret0 = 3;
    let secret1 = 7;
    // console.log('正解: ' + secret0 + secret1); // 遊ぶときは隠します。

    let hit = 0;
    let clearAttempt = 0;

    for (let attempt = 1; attempt <= 5; attempt++) {
        let guess0 = Number(prompt('十の位を入力してね（0〜9）'));
        let guess1 = Number(prompt('一の位を入力してね（0〜9）'));

        let valid0 = guess0 >= 0 && guess0 <= 9;
        let valid1 = guess1 >= 0 && guess1 <= 9;
        let duplicate = guess0 === guess1;

        if (valid0 === false || valid1 === false) console.log('invalid'); // 出力: invalid
        else if (duplicate === true) console.log('重複'); // 出力: 重複
        else {
            hit = 0;
            let blow = 0;
            if (secret0 === guess0) hit = hit + 1;
            if (secret1 === guess1) hit = hit + 1;
            if (secret0 === guess1 && secret0 !== guess0) blow = blow + 1;
            if (secret1 === guess0 && secret1 !== guess1) blow = blow + 1;
            console.log('ヒント: ヒット ' + hit + '、ブロー ' + blow); // 出力: ヒント: ヒット 0、ブロー: 2
            if (hit === 2) {
                clearAttempt = attempt;
                break;
            }
        }
    }

    if (hit === 2) {
        console.log('クリア！' + clearAttempt + '回で当てた！'); // 出力: クリア！3回で当てた！
    } else {
        console.log('残念！正解は ' + secret0 + secret1 + ' でした'); // 出力: 残念！正解は 37 でした
    }
```
</CodePreview>
</Solution>
</Exercise>

ここまでのコードで、2桁のマスターマインドが遊べるようになりました。時間に余裕がある人は、続きの発展問題に取り組んでみましょう。
