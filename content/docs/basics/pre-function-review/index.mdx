### 演習1

<Exercise>

「おはよう」「こんにちは」「こんばんは」の3つを、それぞれ別の行に出力せよ。ただし、出力する文字は変数に入れて（`let` を使い）、文字列の結合を使って出力すること。

<Solution>
<CodePreview>
```javascript
let aisatsu1 = 'おはよう';
let aisatsu2 = 'こんにちは';
let aisatsu3 = 'こんばんは';

console.log(aisatsu1); // 出力: おはよう
console.log(aisatsu2); // 出力: こんにちは
console.log(aisatsu3); // 出力: こんばんは
```
</CodePreview>
</Solution>

</Exercise>

### 演習7

<Exercise>

```javascript
} else {
    console.log('もう一歩');
}
```
</CodePreview>
</Solution>

### 演習8

<Exercise>

for 文を使って次の2つに答えよ。

1. 「3」「4」「5」「6」「7」「8」「9」「10」をこの順でそれぞれ出力せよ。
2. 1〜10 までの整数のうち、2と7以外を小さい順に出力せよ。（`!==` を使うこと）

<Solution>
<CodePreview>
```javascript
// 1. 3から10を出力
for (let i = 3; i <= 10; i++) {
    console.log(i);
}

// 2. 1〜10のうち 2と7以外を出力
for (let i = 1; i <= 10; i++) {
    if (i !== 2 && i !== 7) {
        console.log(i);
    }
}
// 出力: 1, 3, 4, 5, 6, 8, 9, 10
```
</CodePreview>
</Solution>

</Exercise>

### 演習9

<Exercise>

for 文と `break` を使って、1から順に整数を出力せよ。ただし、値が 15 より大きくなったらそこで繰り返しを終了すること。（結果として 1〜15 が出力される）

<Solution>
<CodePreview>
```javascript
for (let i = 1; i <= 20; i++) {
    // 15 を超えたらループを終了する
    if (i > 15) {
        break;
    }
    console.log(i);
}
// 出力: 1, 2, 3, …, 15
```
</CodePreview>
</Solution>

</Exercise>

### 演習10

<Exercise>

for 文と if 文を組み合わせて、1から20までの整数を順に出力せよ。ただし、3の倍数のときは数ではなく「Fizz」と出力すること。（3の倍数とは3で割った余りが0の数。`i % 3` で余りを計算できる）

<Solution>
<CodePreview>
```javascript
for (let i = 1; i <= 20; i++) {
    if (i % 3 === 0) {
        console.log('Fizz');
    } else {
        console.log(i);
    }
}
// 出力:
// 1
// 2
// Fizz
// 4
// 5
// Fizz
// 7
// 8
// Fizz
// 10
// …（以降も同様に20まで）
```
</CodePreview>
</Solution>

</Exercise>

### 演習10-発展

<Exercise>

for 文と if / else if / else を組み合わせて、1から30までの整数を順に出力せよ。ルールは次のとおり。

- 3の倍数のときは「Fizz」
- そうではなく5の倍数のときは「Buzz」
- それ以外はその数

（`i % 3` と `i % 5` を使う。3の倍数を先に判定するので、両方の倍数のときは「Fizz」になる）

<Solution>
<CodePreview>
```javascript
for (let i = 1; i <= 30; i++) {
    if (i % 3 === 0) {
        console.log('Fizz');
    } else if (i % 5 === 0) {
        console.log('Buzz');
    } else {
        console.log(i);
    }
}
```
</CodePreview>

最初の15行の出力は次のようになる。

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
Fizz
```

</Solution>

</Exercise>
