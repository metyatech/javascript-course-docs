ここまでに学んだ文法だけで、コンソールで遊べる「数字あてゲーム」を作ります。

正解の数字は `7` に固定します。プレイヤーが `1`〜`9` の数字を入力すると、正解かどうか、または正解より小さいか大きいかを表示します。最後は、5回まで挑戦できるゲームにします。

今回は、小数やキャンセルは細かく扱いません。入力チェックでは「`1` より小さいか」「`9` より大きいか」だけを確認します。

---

## 作るゲーム

| 正解 | 入力 | 出力 |
| :--: | :--: | :-- |
| `7` | `7` | `正解！` |
| `7` | `3` | `もっと大きい` |
| `7` | `9` | `もっと小さい` |
| `7` | `0` | `1〜9の数字を入力してね` |

最初は、入力した数字を表示するだけの小さなコードから始めます。そこに判定を少しずつ追加して、ゲームらしくしていきましょう。
