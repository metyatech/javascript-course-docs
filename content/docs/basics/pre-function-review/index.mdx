## ヒット＆ブローって何？

ヒット＆ブローは、0〜9の数字を4つ使い、同じ数字をくり返さない並びを当てるゲームです。予想を入力すると、コンピュータがヒットとブローで返事をします。

- ヒット：数字も場所も合っている個数
- ブロー：数字は合っているが、場所が違う個数

正解が `1234` のときは、次のように判定します。

| 予想 | 返事 | 意味 |
| :--: | :-- | :-- |
| `5678` | `0ヒット 0ブロー` | 使われている数字がありません |
| `1256` | `2ヒット 0ブロー` | `1` と `2` が同じ場所にあります |
| `4321` | `0ヒット 4ブロー` | 4つとも数字は合っていますが、場所が違います |
| `1234` | `4ヒット 0ブロー` | クリアです |

### 演習1

<Exercise>

まずは、プレイヤーの予想を受け取れるようにしましょう。正解は今だけ `let secret = '1234';` と固定しておきます。`prompt` で4文字の入力を受け取り、ちょうど4文字で、すべて `0`〜`9` なら `OK`、そうでなければ `invalid` と出力してください。

<Solution>
<CodePreview>
```javascript
    let secret = '1234';
    console.log('正解: ' + secret); // 出力: 正解: 1234
    let guess = prompt('4桁の数字を入力してね');
    let valid = guess.length === 4;
    for (let i = 0; i < 4; i++) {
        if (guess[i] < '0' || guess[i] > '9') valid = false;
    }
    if (valid === true) console.log('OK'); // 出力: OK
    else console.log('invalid'); // 出力: invalid
```
</CodePreview>
</Solution>
</Exercise>

### 演習2

<Exercise>

ヒット＆ブローでは、同じ数字を2回使わないことにします。演習1のコードに、同じ数字が2か所にないかを調べる入れ子の `for` を追加し、`OK`、`invalid`、`重複` のどれかを出力しましょう。

<Solution>
<CodePreview>
```javascript
    let secret = '1234';
    console.log('正解: ' + secret); // 出力: 正解: 1234
    let guess = prompt('4桁の数字を入力してね');
    let valid = guess.length === 4;
    let duplicate = false;
    for (let i = 0; i < 4; i++) {
        if (guess[i] < '0' || guess[i] > '9') valid = false;
        for (let j = i + 1; j < 4; j++) {
            if (guess[i] === guess[j]) duplicate = true;
        }
    }
    if (valid === false) console.log('invalid'); // 出力: invalid
    else if (duplicate === true) console.log('重複'); // 出力: 重複
    else console.log('OK'); // 出力: OK
```
</CodePreview>
</Solution>
</Exercise>

### 演習3

<Exercise>

次は、予想に対してヒットとブローを数えます。入力が正しく、重複もないときだけ、`guess` の各桁を `secret` の各桁と比べて判定してください。

<Solution>
<CodePreview>
```javascript
    let secret = '1234';
    console.log('正解: ' + secret); // 出力: 正解: 1234
    let guess = prompt('4桁の数字を入力してね');
    let valid = guess.length === 4;
    let duplicate = false;
    for (let i = 0; i < 4; i++) {
        if (guess[i] < '0' || guess[i] > '9') valid = false;
        for (let j = i + 1; j < 4; j++) {
            if (guess[i] === guess[j]) duplicate = true;
        }
    }
    if (valid === false) console.log('invalid'); // 出力: invalid
    else if (duplicate === true) console.log('重複'); // 出力: 重複
    else {
        let hit = 0;
        let blow = 0;
        for (let i = 0; i < 4; i++) {
            for (let j = 0; j < 4; j++) {
                if (i === j && secret[i] === guess[i]) hit = hit + 1;
                if (i !== j && secret[j] === guess[i]) blow = blow + 1;
            }
        }
        console.log('ヒント: ヒット ' + hit + '、ブロー ' + blow); // 出力: ヒント: ヒット 4、ブロー 0
    }
```

### 演習4

<Exercise>

最後に、10回まで予想できるゲームにします。演習3までの入力チェックと判定を `for` で10回くり返し、ヒットが4になったら何回目で当たったかを保存して `break` してください。`console.log('正解: ' + secret);` はコメントアウトして、プレイヤーには答えが見えないようにしてください。

<Solution>
<CodePreview>
```javascript
    let secret = '1234';
    // console.log('正解: ' + secret); // プレイヤーには見せない
    let clearAttempt = 0;
    let hit = 0;
    for (let attempt = 1; attempt <= 10; attempt++) {
        let guess = prompt('4桁の数字を入力してね');
        let valid = guess.length === 4;
        let duplicate = false;
        hit = 0;
        let blow = 0;
        for (let i = 0; i < 4; i++) {
            if (guess[i] < '0' || guess[i] > '9') valid = false;
            for (let j = i + 1; j < 4; j++) {
                if (guess[i] === guess[j]) duplicate = true;
            }
        }
        if (valid === false) console.log('invalid'); // 出力: invalid
        else if (duplicate === true) console.log('重複'); // 出力: 重複
        else {
            for (let i = 0; i < 4; i++) {
                for (let j = 0; j < 4; j++) {
                    if (i === j && secret[i] === guess[i]) hit = hit + 1;
                    if (i !== j && secret[j] === guess[i]) blow = blow + 1;
                }
            }
            console.log('ヒント: ヒット ' + hit + '、ブロー ' + blow); // 出力: ヒント: ヒット 4、ブロー 0
            if (hit === 4) { clearAttempt = attempt; break; }
        }
    }
    if (hit === 4) console.log('クリア！' + clearAttempt + '回で当てた！'); // 出力: クリア！1回で当てた！
    else console.log('残念！正解は ' + secret + ' でした'); // 出力: 残念！正解は 1234 でした
```
</CodePreview>
</Solution>
</Exercise>
