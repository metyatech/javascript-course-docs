## 文字列の結合

`+` 演算子を使うと、2つの文字列を1つにつなげられます。

```js
"文字列1" + "文字列2"
```

<CodePreview>
```javascript
let firstName = "山田";
let lastName = "太郎";

// 文字列どうしを結合する
console.log(firstName + lastName); // 出力: 山田太郎
```
</CodePreview>

### 数値と文字列の結合

数値と文字列を `+` でつなぐと、数値が文字列に変換されてから結合されます。

<CodePreview>
```javascript
// 円周率
let pi = 3.14;

console.log("円周率は約" + pi);        // 出力: 円周率は約3.14
console.log("得点は" + 80 + "点です"); // 出力: 得点は80点です
```
</CodePreview>

## 型（タイプ）

**型**とは、値の種類のことです。JavaScriptには主に次の型があります。

| 型 | 例 |
| :-: | :-: |
| 文字列（String） | `"あいうえお"` |
| 数値（Number） | `123`、`11.7` |
| 真偽値（Boolean） | `true`、`false` |
| 未定義（Undefined） | — |
| 空（Null） | — |
| オブジェクト（Object） | — |

```js
let text  = "hello"; // 文字列 (String)
let score = 95;      // 数値 (Number)
let flag  = true;    // 真偽値 (Boolean)
```

## prompt

`prompt` は、**ユーザーに何かを入力させたいとき**に使います。

```js
prompt(メッセージ)
```

- ユーザーが **OK** を押すと、入力内容が**文字列型**で返ってきます。
- ユーザーが **キャンセル** を押すと `null` が返ってきます。

<CodePreview>
```javascript
// 入力内容を変数 answer に入れる
let answer = prompt("お名前を入力してね");

// 入力した名前を使って出力する
console.log("こんにちは、" + answer + "さん！"); // 出力例: こんにちは、田中さん！
```
</CodePreview>

## 型変換：文字列 → 数値

`prompt` の入力値は常に**文字列型**です。計算に使うには、`Number()` で数値型に変換する必要があります。

```js
Number(文字列)
```

<CodePreview>
```javascript
// 入力は文字列型 → "3" のような文字列が入ってくる
let input = prompt("数値を入力してください");

// このまま足し算すると文字列結合になってしまう
console.log(input + 10); // 出力例: 310 （文字列の "3" と 10 がくっつく）

// Number() で数値型に変換してから計算する
let num = Number(input);
console.log(num + 10);   // 出力例: 13
```
</CodePreview>
